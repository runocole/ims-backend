Django-based business management solution integrating CRM, inventory control, sales tracking, and automated invoice generation. Features real-time stock monitoring, customer interaction tracking, sales pipeline visualization, and professional PDF invoice generation with ReportLab. Containerized with Docker and deployed on Hostinger VPS with Gunicorn, utilizing PostgreSQL for data persistence, Redis for caching, and Pandas for Excel/CSV import/export. The system provides RESTful endpoints with Django templating, enabling businesses to manage customers, track inventory, process sales orders, and generate audit trails through an intuitive dashboard interface.
